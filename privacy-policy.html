Условия использования сервиса во время бета-теста

Используя app.llamagraphic.ru (далее — сервис) во время бета-теста, вы соглашаетесь с этими условиями.

Использование сервиса бесплатное. Количество доступных инфографик для формирования не ограничено.

Сформированные инфографики хранятся на сервере в РФ, они могут быть доступны администрации сервиса.

Сервис не предоставляет никаких гарантий доступности сервиса и сформированных ранее инфографик.

Администрация не гарантирует никаких сроков обновления сервиса и добавления новых возможностей.

Администрация сайта не контролирует и не несет ответственность за сайты третьих лиц, на которые пользователь может перейти по ссылкам.

Какая информация подлежит сбору

Сбору подлежат только сведения, обеспечивающие возможность поддержки обратной связи с пользователем. Некоторые действия пользователей автоматически сохраняются в журналах сервера:

  • IP адрес;
  • информация из cookies;
  • информация о браузере (или иной программе, которая осуществляет доступ к показу рекламы);
  • время доступа;
  • адрес страницы, на которой расположен рекламный блок;
  • реферер (адрес предыдущей страницы).

При авторизации при помощи Google сайт получает доступ к имени, псевдониму и фотографии из вашего профиля.

При авторизации при помощи Яндекс сайт получает доступ к имени, фамилии, псевдониму и аватару из вашего профиля.

Как используется полученная информация

Сведения, предоставленные пользователем, используются для персонализации работы сервиса.

  • шифрование архивных копий ресурса;
  • регулярные проверки на предмет наличия вредоносных кодов;
  • использование для размещения сайта виртуального выделенного сервера.
